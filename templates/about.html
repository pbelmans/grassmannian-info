How to contribute

It is to be expected that there are typos, mistakes, or things that should just be better. If that is the case you can

  • contact me (see below)
  • create an issue on GitHub

How to cite?

If you use biblatex: 
@online{grassmannian,
  author = {Belmans, Pieter},
  title = {Grassmannian.info --- A periodic table of (generalised) Grassmannians},
  url = {https://grassmannian.info},
  year = {% raw %}{{% endraw %}{{ now.year }}{% raw %}}{% endraw %},
}

If you still use bibtex: 

@misc{fanography,
  author = {Belmans, Pieter},
  title = {Grassmannian.info --- A periodic table of (generalised) Grassmannians},
  howpublished = {% raw %}{\url{https://grassmannian.info}}{% endraw %},
  year = {% raw %}{{% endraw %}{{ now.year }}{% raw %}}{% endraw %},
}
